플러터 -> 구굴의 오픈소스 모바일 앱 개발 프레임 워크, 고품질 기본 인터페이스를 제작하는데 도움을 주는 크로스 플랫폼 프레임워크
--

특징
1. 개발 언어로 dart 사용
2. 뛰어난 성능 -> react-native 의 경우 자바 스크립트로 네이티브와 브릿지를 통해 통신하는 반면 flutter는 직접 컴파일 되어서 직접 render하기 때문에 빠름
3. 풍부하고 유연한 디자인 - 작성 가능한 위젯 세트, 풍부만 애니메이션 라이브러리 및 확장 가능한 계층형 아키텍처를 제공
4. 고품질 환경 - 이식성 있는 GPU 가속 렌더러 및 고성능의 네이티브 ARM 코드 런타임, 플랫폼 상호 운용성 기능을 통해 기기 및 플랫폼 전반에 걸쳐 고품질 환경을 지원한다.
5. 신속한 개발 - 상태를 기록하는 핫리로드, 새로운 반응형 프레임워크, 다양한 위젯 세트 및 통합 도구와 같은 기능 제공
6. 강력하고 쉬운 크로스플랫폼





위젯 -> 자체적 위젯을 사용하기에 좋은 룩앤필 빠른 속도 커스터마이즈 확장 가능성을 지원


성능 -> 문맥 교환하는 javascript 브릿지를 거치지 않고 플랫폼과 직접적으로 커뮤니케이션 한다.

Hot reload -> 디버깅 중인 에뮬레이터를 종료하지 않아도 코드상 수정한 내용이 바로 적용 되어 개발 효율을 극대화
