##프로젝트 폴더는 기본적으로 자동으로 만들어지는데

1. android - 안드로이드 앱 구성
2. ios - IOS앱 구성
3. lib - dart 파일 - 기능 구현
4. test - 테스트 다트 파일 -> 테스트 코드를 별도 폴더에 구성해 앱을 빌드할 때 포함하지 않게 할 목적으로 제공

##프로젝트 파일 구성 알아보기
빌드 도구가 자동으로 관리되므로 개발자가 신경쓰지 않아도 됨.
1. lib/main.dart - 앱의 메인 다트 파일
2. .gitignore - git에 업로드하지 않을 파일 등록
3. pubspec.yarml - 플러터 프로젝트의 메인 환경 파일
